Component: TestDisseminationAppC

configuration TestDisseminationAppC

TestDisseminationAppC exercises the dissemination layer, by causing the node with ID 1 to inject 2 new values into the network every 4 seconds. For the 32-bit object with key 0x1234, node 1 toggles LED 0 when it sends, and every other node toggles LED 0 when it receives the correct value. For the 16-bit object with key 0x2345, node 1 toggles LED 1 when it sends, and every other node toggles LED 1 when it receives the correct value. See TEP118 - Dissemination for details.
