Component: TestAdcAppC

configuration TestAdcAppC

Tests the AdcC subsystem and switches on leds 0, 1 and 2 if the test is successful: LED0 denotes a successful Read operation, LED1 denotes a successful ReadNow operation, LED2 denotes a successful ReadStream operation. Requires a platform-specific DemoSensorC component that provides Read, a platform-specific DemoSensorStreamC component that provides ReadStream and a platform-specific DemoSensorNowC component that provides ReadNow and Resource
